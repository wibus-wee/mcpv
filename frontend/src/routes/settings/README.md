# Routes/Settings

Settings 相关子路由目录。
包含 Core Connection、Runtime、SubAgent、Appearance、Gateway 与 Advanced 设置页。
每个路由文件仅负责装配页面组件。

## Files

- **index.tsx**: Settings 首页路由
- **appearance.tsx**: Appearance 设置路由
- **core-connection.tsx**: Core Connection 设置路由
- **runtime.tsx**: Runtime 设置路由
- **gateway.tsx**: Gateway 设置路由
- **subagent.tsx**: SubAgent 设置路由
- **advanced.tsx**: Advanced 设置路由（更新检查、遥测、托盘、诊断导出）
