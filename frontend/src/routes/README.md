# Routes

TanStack Router 的文件路由目录。
顶层文件对应路由入口，`__root.tsx` 提供统一布局。
子目录用于嵌套路由。

## Files

- **__root.tsx**: 根布局，包含 providers、sidebar 与 outlet
- **index.tsx**: Dashboard 路由入口
- **about.tsx**: About 路由入口
- **servers.tsx**: Servers 路由入口
- **logs.tsx**: Logs 路由入口
- **topology.tsx**: Topology 路由入口
- **settings.tsx**: Settings 路由入口
- **settings/**: Settings 子路由目录
