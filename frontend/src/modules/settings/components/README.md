# Settings Components

Settings 页面专用 UI 组件。
包含运行时与 SubAgent 的卡片布局与头部组件。
仅负责展示与表单布局。

## Files

- **runtime-settings-card.tsx**: Runtime 设置卡片与表单布局
- **settings-header.tsx**: Settings 页面头部区域
- **subagent-settings-card.tsx**: SubAgent 设置卡片与表单布局（包含标签变更埋点）

## Directories

- **settings-card/**: SettingsCard 复合组件与字段封装
