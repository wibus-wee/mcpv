# Servers Lib

Servers 模块的工具与文案配置集合。
包含导入解析、配置重载与表单文案帮助等基础能力。
该目录仅承载无 UI 的逻辑与常量。

## Files

- **mcp-import.test.ts**: MCP JSON 导入解析的测试
- **mcp-import.ts**: MCP JSON 导入解析逻辑
- **reload-config.ts**: 触发配置热重载的辅助函数
- **server-form-content.ts**: 服务器表单文案、帮助与建议规则
