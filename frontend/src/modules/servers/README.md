# Modules/Servers

统一的服务器管理模块。
提供服务器列表、详情、工具与配置编辑等能力。
页面以主从布局组织，并配合运行态与工具状态展示。

## Files

- **servers-page.tsx**: Servers 主页面与整体布局
- **atoms.ts**: 服务器选择状态的 jotai atoms
- **hooks.ts**: 服务器与运行态数据获取 hooks

## Directories

- **components/**: Servers 模块 UI 组件
- **lib/**: Servers 模块工具与文案配置
