# Exec Plans

该目录存放可执行规划文档，用于指导跨模块或复杂变更的落地。
每个计划独立自洽，覆盖目的、步骤、验证与回滚方式。
修改或新增 ExecPlan 时必须同步更新此清单。

- **EXEC_PLAN_AGGREGATION_REFRESH.md**: Aggregation 刷新策略的执行计划
- **EXEC_PLAN_ASYNC_SERVER_INIT.md**: 异步 Server 初始化管理的执行计划

- **EXEC_PLAN_IMPORT_MCP_JSON.md**: MCP JSON 导入能力的执行计划
